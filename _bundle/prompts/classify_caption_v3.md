아래 인스타그램 쇼핑 영상 목록을 분석해서 JSON 배열만 반환해. 다른 말 금지.

각 항목에서 추출:
- topic: 이 영상이 다루는 구체적인 소재명 (예: "스팀 청소기", "접이식 식탁", "된장찌개 레시피")
- product_type: 이 제품을 쿠팡에서 어느 카테고리로 사야 하는지를 결정하는 핵심 명사구.
  표면 단어가 아니라 **본문(캡션+대본) 전체에서 읽은 실제 기능과 사용 맥락**으로 정한다.
  - 실제 기능으로 판단(표면 모양·이름에 속지 말 것):
    퍼즐 모양이어도 본문이 난방·바닥·층간소음이면 "바닥 매트"(완구 아님)
  - 카테고리를 가르는 사용 대상/장소가 본문에 드러나면 반드시 포함:
    차량에 쓰면 "차량용 발수코팅제", 욕실 거울에 쓰면 "욕실 유리 코팅제", 식용유 담으면 "오일 디스펜서"
  - 본문에 그런 맥락이 없으면 억지로 넣지 말고 일반 명사로 둔다 (식탁, 청소기처럼)
  - 브랜드·색상·사이즈처럼 카테고리와 무관한 수식어는 제외
  - **식품·식재료·건강식품도 상품이다**: 음식 판매·레시피·먹방 영상에서 핵심 식품명을 product_type으로 뽑는다 (예: 복숭아, 잡채 재료, 단백질 케이크, 골든밀크 재료, 다진마늘). "미상"으로 버리지 말 것.
- is_recipe: 요리법·레시피·만드는 방법이 주 내용이면 true, 아니면 false

레시피 판단 기준:
- 재료 목록, 조리 순서, 계량(g/ml/큰술 등)이 포함되어 있으면 true
- 단순 음식 사진, 식당 소개, 식품 판매이면 false

제품/쇼핑 상품 판단:
- 식품·식재료·건강식품은 그 자체가 상품이다. 쇼핑 상품이 전혀 없으면 topic과 product_type 모두 "미상".
- 제품명을 숨긴 생활 노하우·팁 게시물이라도 특정 제품이 핵심 도구로 반복되면 그 제품군을 추출.
  제품이 전혀 식별되지 않으면 "미상"

반드시 아래 형식의 JSON 배열만 반환:
[{"id": "...", "topic": "...", "product_type": "...", "is_recipe": false}, ...]
